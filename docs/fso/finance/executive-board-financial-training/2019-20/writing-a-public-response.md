# Writing a public purpose

- All purchases require a public purpose (i.e. a justification for the purchase), which are also public record. Each public purpose you write must include:
  - _who made the purchase_
  - _what the purchases were for_
  - _how do the purchases benefit ASU and Fulton_
- A STRONG public purpose statement looks like this:
  -  **Who are you:** Full Fulton student organization name
  -  **What** you purchased
  -  **Why** does ASU/Fulton care? How did this make an impact?
  - _Example: Funky Fulton Engineers are purchasing electronics to attend the FFE Competition to bring prestige and recognition to ASU._
