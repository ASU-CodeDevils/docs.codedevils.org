Flameboi - Slack API
====================
<: <img alt="resful" src="https://www.qa.codedevils.org/public/media/flameboy.png" width="500"> |
Flameboi is CodeDevils' Slack application used primarily to add new users to the Slack space and send
notifications/welcome messages. Flameboi is shared on the ASU Slack Enterprise grid, so any organization that wants to
add the application will be able to.

To add it, click the <a href="https://slack.com/oauth/authorize?client_id=90244543126.683893585046&scope=admin.conversations:read,bot,incoming-webhook,admin.apps:write"><img alt="Add to Slack" class="no-shadow" width="70" src="https://platform.slack-edge.com/img/add_to_slack.png" srcset="https://platform.slack-edge.com/img/add_to_slack.png 1x, https://platform.slack-edge.com/img/add_to_slack@2x.png 2x"></a>
button. We are still working on making this application easier to distribute, but for right now Flameboi is used solely
for our organization.
:>

You can see the GitHub documentation [here](https://asu-codedevils.github.io/flameboi-slack-api/index.html) if you are
developing the Flameboi API.
