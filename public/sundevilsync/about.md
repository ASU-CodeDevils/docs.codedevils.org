About
=====
The [organization about dashboard](https://asu.campuslabs.com/engage/actioncenter/organization/codedevils/about) contains
all the public information about CodeDevils. Everything on this page is displayed on the [organization home page](https://asu.campuslabs.com/engage/organization/codedevils)
and is used to connect students with CodeDevils.

_Note: If you update any of these fields, do not forget to select the **UPDATE** at the bottom of the page._

### Updating the Profile Picture
You can update the profile picture simply by selecting _Choose File_ below the image. You can then select the image from
your desktop that you want to be the profile picture. It will crop the picture accordingly to fit the avatar bubble.
