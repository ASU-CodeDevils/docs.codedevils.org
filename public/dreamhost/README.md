DreamHost
=========

DreamHost is the CodeDevils' web hosting service. It not only hosts the
website, but provides our email, database, and documentation. For more
information visit the [DreamHost website](https://www.dreamhost.com/),
or to get hosting checkout the [DreamHost special web hosting](https://www.dreamhost.com/promo/codedevils/)
for exclusive CodeDevils discounts.
