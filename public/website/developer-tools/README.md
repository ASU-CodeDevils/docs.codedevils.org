Developer Tools
===============
CodeDevils prides itself in being ASU's leader in student software innovation. Our website hosts not only the landing
page for our organization, but hosts a variety of software tools.

**Coming soon!** Our RESTful APIs provide a series of endpoints used to alter/obtain data from CodeDevils. Check out 
[this RESTful API tutorial](https://restfulapi.net/) to learn more. :>

## GraphQL
<: <img alt="graphql" src="https://upload.wikimedia.org/wikipedia/commons/thumb/1/17/GraphQL_Logo.svg/2000px-GraphQL_Logo.svg.png" width="600"> | 
**Coming soon!** GraphQL is a query language developed by Facebook that aims to replace RESTful APIs. It provides a more
robust and dynamic means of obtaining data by replacing multiple endpoints standard RESTful APIs require with just one. 
Check out [the GraphQL website](https://graphql.org/) to learn about GraphQL, or take [this tutorial](https://www.howtographql.com/)
to learn how to use it. :>

## Slack API
<: <img alt="resful" src="https://a.slack-edge.com/80588/marketing/img/icons/icon_slack_hash_colored.png" width="500"> |
**Coming soon!** Slack applications allow CodeDevils to interact with Slack programmatically, enabling us to add members
during registration, post to channels and send welcome letters to new CodeDevils members. Check out [the Slack API](https://api.slack.com/)
to learn how to develop a Slack application. :>
